
# Drum Kit
A responsive Drum Kit where you can play drum sounds by clicking or typing the keys mentioned on each drum.




## Run Locally

Clone the project

```bash
  git clone https://github.com/Harshit-Khattar/DrumKit
```

Go to the project directory

```bash
  cd my-project
```
